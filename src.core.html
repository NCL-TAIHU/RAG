class src.core.filter.Filter[source]

Bases: BaseModel

Base class for statically declared filters. Filters are dependent on specific document types and are used to filter search results. Subclasses should override must_fields() and filter_fields() manually. All fields should be Optional[List[…]] even if the underlying document field is singular. This unifies the semantics: all filters represent set-based inclusion or exclusion logic.
